Play 2.0 설치

전제 조건

Play framework의 실행에는 JDK 6 이상 이 필요합니다.

MacOS를 사용하면 Java는 사전 설치되어 있습니다. Linux를 사용하는 경우는 Sun JDK 또는 OpenJDK를 설치 (많은 Linux 배포판 표준이있다 gcj 아님)하십시오. Windows를 사용하는 경우, 최신의 JDK 패키지를 다운로드하여 설치하십시오.

현재 경로에서 java 와 javac 명령을 사용할 수 있는지 확인하십시오. (이것은 프롬프트에서 java-version 과 javac-version 을 입력하여 확인할 수 있습니다.)

바이너리 패키지 다운로드

Play 2.0 바이너리 패키지 (최신 2.0 RC)를 다운로드하여 작성 · 읽기 권한이있는 위치에 배포하십시오. (Play 중에 파일이 아카이브 디렉토리에 기록되므로 / opt 에 설치하는 것이 아니라, / usr / local 또는 쓰기 권한이있는 위치에 설치하십시오)

play 스크립트에 경로를 통과

편의를 위해, Play 설치 디렉토리를 PATH 환경 변수에 추가 둡시다. 예를 들어, Unix 계열의 OS를 사용하는 경우 다음과 같이 설정합니다 :

export PATH = $ PATH : / path / to / play20

Windows에서는 시스템 환경 변수를 설정해야합니다. 이것은 환경 변수로 설정되어있는 경로를 업데이트하는 것을 의미하고 공백이있는 경로를 사용하지 않도록하십시오.

UNIX 계열의 OS를 사용하는 경우에는 play 스크립트가 실행되어 있는지 확인하십시오 (그렇게되어 있지 않으면 chmod a + x play 를 실행하십시오)

play 명령의 시작 확인

쉘 play help 명령을 실행합시다.

$ play help

설치가 성공적으로 연결하면, 도움말이 표시되어야합니다.
