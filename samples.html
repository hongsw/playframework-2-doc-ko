샘플 응용 프로그램

Play 2.0 패키지에는 Java와 Scala 모두에서 작성된 샘플 응용 프로그램의 포괄적 인 세트를 제공합니다. 이것은 코드 조각 및 예제를 찾는 데 매우 좋은 장소입니다.

샘플 응용 프로그램은 Play 2.0 패키지 samples / 디렉토리에 포함되어 있습니다.

Hello world

이것은 Play 2.0의 기본 기능을 소개하는위한 기본 응용 프로그램입니다.

  • 컨트롤러와 액션
  • 라우팅 및 리버스 라우팅
  • 정적 콘텐츠에 대한 링크
  • 템플릿 엔진 사용
  • 양식 제출과 검증

Computer database

이것은 JDBC 데이터베이스를 이용한 전통적인 CRUD 애플리케이션입니다.

  • JDBC 데이터베이스에 대한 액세스. Java 버전에서는 Ebean, Scala 판에서는 Anorm를 이용합니다.
  • 테이블의 페이징과 CRUD 용 양식
  • CSS 프레임 워크 ( Twitter Bootstrap ) 내장.

Twitter Bootstrap을 이용하기 위해서는, Play 2.0 표준 양식 헬퍼를 생성하는 input 요소의 레이아웃과 다른 레이아웃이 필요합니다. 따라서이 프로그램은 폼 헬퍼를 정의하고 있습니다.

양식

이것은 몇 가지 전형적인 양식의 사용 방법을 제시하는 더미 응용 프로그램입니다.

  • 복잡한 양식과 검증
  • 동적 인 반복 값 양식 제출

ZenTasks

이 고기능 TODO 목록은 최근 유행 Ajax 기반의 web 어플리케이션의 데모입니다. 아직 미완성 앞으로 더 기능이 추가 될 예정입니다. 현재 다음과 같은 기능의 도움이됩니다.

  • 보안 및 인증
  • Ajax의 이용과 JavaScript의 리버스 라우팅
  • LESS CSS와 CofeeScript 등 트랜스 컴파일 된 자산의 이용

CometClock

이것은 forever-frame 말하는 기술을 통해 클럭 이벤트 서버에서 웹 브라우저에 푸시하는 매우 간단한 Comet의 데모입니다.

  • Comet 연결 만들기
  • Akka의 Actor 사용 (Java 버전)
  • Enumerators의 개조 (Scala 버전)

WebSocket 채팅

이 응용 프로그램은 WebSocket을 사용하여 구축 된 방입니다. 게다가 같은 방에서 이야기 bot도 있습니다.

  • WebSocket 연결
  • 고급 Akka 이용

Comet 모니터링

이 모바일 웹 응용 프로그램은 Play 서버의 성능을 모니터링합니다.

  • Enumerator와 Enumeratee 고급 사용
