Play 2.0 모듈

여기는 모듈 목록이 Play 웹 사이트에 등록 할 수있게 될 때까지의 임시 공개 장소입니다.

모듈을 작성하기위한 단계를 설명했다 블로그 기사가 objectify.be 하면 developer.vz.net 있습니다.

Authentication and Authorization 모듈 (Scala)

  • Web 사이트 : [https://github.com/t2v/play20-auth]
  • 문서 (영어) : [https://github.com/t2v/play20-auth/blob/master/README.md]
  • 문서 (일본어) : [https://github.com/t2v/play20-auth/blob/master/README.ja.md]
  • 설명 : 인증 및 권한 부여 메커니즘을 제공하는 모듈입니다.

Authenticity Token 모듈

  • Web 사이트 : https://github.com/orefalo/play2-authenticitytoken
  • 문서 : https://github.com/orefalo/play2-authenticitytoken/blob/master/README.md
  • 설명 : Play1 있었다 authenticity token을 재현하는 모듈입니다. CSRF 공격에 대한 대책이 있습니다.

Deadbolt 2 플러그인

  • Web 사이트 (문서, 샘플) : https://github.com/schaloner/deadbolt-2
  • 설명 : Deadbolt는 컨트롤러의 메소드와 뷰의 일부 간단한 AND / OR / NOT의 문법에서 액세스 권한을 정의하는 것으로, 인증 방식을 제공하는 플러그인입니다.

Dust 플러그인

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins/tree/master/dust
  • 설명 : dust는 클라이언트 사이드의 템플릿 언어 지원을 추가하는 플러그인입니다.

Groovy Templates 플러그인

  • Web 사이트 : https://github.com/manuelbernhardt/play2-groovy-templates
  • 문서 : https://github.com/manuelbernhardt/play2-groovy-templates/blob/master/README.md
  • 설명 : Play1의 Groovy 템플릿을 Play 2에서 사용할 수있는 플러그인입니다.

Groovy Templates 플러그인 - gt-engine-play2

  • Web 사이트 : https://github.com/mbknor/gt-engine-play2
  • 문서 : https://github.com/mbknor/gt-engine-play2/blob/master/README.markdown
  • 설명 : Play 1 Groovy 템플릿을 Play 2에서 사용할 수있는 플러그인입니다. Play 1 "Faster Groovy Templates"모듈 http://www.playframework.org/modules/fastergt 에서 사용되는 것과 동일한 gt-engine을 내부적으로하고 있습니다.
  • 샘플 : https://github.com/mbknor/gt-engine-play2/tree/master/samples

Guice 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins
  • 설명 : Guice에 따르면 DI를 제공하는 플러그인입니다.

HTML5 Tags 모듈

  • Web 사이트 : https://github.com/loicdescotte/Play2-HTML5Tags
  • 문서 : https://github.com/loicdescotte/Play2-HTML5Tags/blob/master/README.md
  • 설명 : Play 2.0 템플릿 클라이언트 측 검증 (required, regex 패턴, max | min lentgh 등)과 특수 입력 필드 (date, email, url, telephone number, ...)의 태그를 사용할 수 있도록하는 모듈입니다.

Japid 모듈

  • Web 사이트 : https://github.com/branaway/japid42
  • 문서 : https://github.com/branaway/japid42
  • 설명 : Japid Java 템플릿 Play 2에서 사용할 수 있도록하는 모듈입니다.

Manual Dependency Injection 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins
  • 설명 : 수동 주입에 의한 DI를 제공하는 플러그인입니다.

MongoDB Jackson Mapper 플러그인 (Java)

  • Web 사이트 (문서, 샘플) : https://github.com/vznet/play-mongo-jackson-mapper
  • 설명 : MongoDB 액세스 및 Jackson 주석의 객체 매핑을 제공하는 플러그인입니다.

MongoDB Salat, Casbah 플러그인 (Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/leon/play-salat
  • 설명 : Salat 및 Casbah에 따르면 MongoDB 액세스와 객체 매핑을 제공하는 플러그인입니다.

Play Mini (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play2-mini
  • 설명 : (표준 SBT 프로젝트) Play를 HTTP 라이브러리로 사용하는 샘플입니다.

Redis 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins
  • 설명 : Redis를 이용한 캐시 API의 구현입니다. Redis 특유의 API도 사용할 수 있습니다.

Emailer 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins
  • 설명 : Apache commons-email에 따르면 emailer을 제공하는 플러그인입니다.

Roy Compiled Asset 플러그인 (Ray)

  • Web 사이트 : https://github.com/pufuwozu/ray
  • 블로그 기사 : http://brianmckenna.org/blog/ray
  • 설명 : Roy 파일을 JavaScript로 컴파일합니다.

Snapshot 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/vznet/play-snapshot
  • 설명 : HtmlUnit을 이용하여 여기 에 Google에 의해 설명되는 Hashbang Snapshot 기능을 제공합니다.

Statsd 플러그인 (Java 및 Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/vznet/play-statsd
  • 설명 : statsd 클라이언트를 제공합니다. client

Typesafe util 플러그인 (Scala)

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins
  • 설명 : 요청 헤더 보안 기능과 플러그인을 관리하기위한 구문 설탕을 제공합니다.

Typesafe SbtGoodies 플러그인

  • Web 사이트 (문서, 샘플) : https://github.com/typesafehub/play-plugins/sbtgoodies
  • 설명 : 추가 sbt 명령을 제공합니다.

WAR 플러그인

  • Web 사이트 : https://github.com/dlecan/play2-war-plugin
  • 문서 : https://github.com/dlecan/play2-war-plugin/blob/develop/README.md
  • 설명 : Play 2.x 응용 켓 옵션을 표준 WAR에 포장합니다.

XForward 모듈

  • Web 사이트 * : https://github.com/orefalo/play2-xforward
  • 문서 : https://github.com/orefalo/play2-xforward/blob/master/Readme.md
  • 설명 : Play1 있었다 Proxy 포워딩에 대한 설정을 제공하는 모듈입니다.

Spring 모듈

  • Web 사이트 : https://github.com/scott-phillips/Spring4Play2
  • 문서 : https://github.com/scott-phillips/Spring4Play2/blob/master/README.markdown
  • 설명 : Play1 있었다 Spring 모듈 Play 2.0 용으로 업데이트했습니다.

JsMessages

  • Web 사이트 : https://github.com/julienrf/play-jsmessages
  • 설명 : 클라이언트 측에서 메시지를 지역화합니다.

Memcached 플러그인

  • Web 사이트 : https://github.com/mumoshu/play2-memcached
  • 설명 : memcached에 따르면 Cache API의 구현입니다.

Stylus 플러그인

  • Web 사이트 : https://github.com/knuton/play-stylus
  • 설명 : Stylus 에 따르면 CSS 컴파일합니다.

ScalikeJDBC 플러그인 (Scala)

  • Web 사이트 : https://github.com/seratch/scalikejdbc-play-plugin
  • 설명 : 또 하나의 Play20위한 데이터베이스 액세스 API입니다.
