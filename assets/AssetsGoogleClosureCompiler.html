Google Closure Compiler를 사용

Closure Compiler 는 JavaScript의 다운로드 실행 속도를위한 도구입니다. Closure Compiler는 그 이름과 같이 컴파일러입니다. 그러나 소스 코드를 기계어로 컴파일하는 것이 아니라 JavaScript를 좋은 JavaScript에 컴파일합니다. JavaScript 코드는 퍼스 분석되어 불필요한 코드를 제거하고 필요에 따라 다시 작성하여 결국 minify됩니다.

app / assets 디렉토리 아래의 JavaScript는 모든 Google Closure Compiler에 의해 처리되고, 오류 및 의존성 체크가됩니다. 또한 빌드 설정에서 활성화되어있는 경우 minify됩니다.

JavaScript의 온 전성 검사

JavaScript 코드는 compile 명령의 실행하거나, 코드 변경시 자동으로 컴파일됩니다. 다른 코드의 컴파일 오류뿐만 아니라 JavaScript 컴파일 오류도 브라우저에서 확인할 수 있습니다.

CommonJS 스타일의 의존 모듈 정의

Play에 내장 된 Closure Compiler는 CommonJS 스타일로 작성된 의존 모듈도 해결 해줍니다. 이것은 RequireJS 에 비슷합니다.

예를 들어, 다음과 같은 lib.js 파일이 있다고합니다.

/ / The lib

Function sum ( a , b )  { 
    return A + b ; 
} 
exports . sum = sum ;

또한 다음과 같은 test.js 파일이 있다고합니다.

/ / The test

require ( "lib" );

Function showSum ( first , second )  { 
    alert ( require ( "lib" ) sum ( first , second )); 
}

showSum ( 2 , 3 ],  4 );

컴파일 후 test.js 의 내용은 다음과 같습니다.

require . register ( "lib" ,  function ( module , exports ,  require ) {  
/ / The lib

Function sum ( a , b )  { 
    return A + b ; 
} 
exports . sum = sum ;

}

require ( "lib" )

/ / The test

Function showSum ( first , second )  { 
    alert ( require ( "lib" ) sum ( first , second )); 
}

showSum ( 2 , 3 ],  4 );

축소

. js 가 . min.js 에 대체 된 축소판 파일도 생성됩니다. 이번 예제의 경우 미리는 test.min.js 되고있을 것입니다. 일반 파일 대신에 축소판을 사용하려면, HTML 파일에서 script의 source 속성을 변경해야합니다.

진입 점

기본적으로 파일 이름 앞에 밑줄 않은 JavaScript 파일은 컴파일됩니다. 이 행동 뜻은 project / Build.scala 에서 javascriptEntryPoints 키를 덮어 써서 변경할 수 있습니다. 이 키는 PathFinder 을 보유하고 있습니다.

예를 들어, app / assets / javascripts / main 디렉토리의 . js 파일 만 컴파일하려면 다음과 같이합니다 :

val main =  PlayProject ( appName , appVersion , mainLang = SCALA ). settings ( 
   javascriptEntryPoints << = baseDirectory ( base  => 
      base  /  "app"  /  "assets"  /  "javascripts"  /  "main"  **  "*. js" 
   ) 
)

기본 정의는 다음과 같습니다 :

javascriptEntryPoints << =  ( sourceDirectory in  Compile ) ( base  => 
   (( base  /  "assets"  **  "*. js" )  ---  ( base  /  "assets"  **  "_ *" )). get 
)
