Play 2.0 문서

Play 2.0은 현대의 web 어플리케이션 개발에 필요한 구성 요소 및 API를 통합 한 생산성 Java와 Scala의 web 어플리케이션 프레임 워크입니다.

Play의 특징은 경량, 무국적, web 친절한 아키텍처 인 기능 예측의 용이성입니다. 또한 Iteratee IO를 기반으로 한 리 엑티브 모델 덕분에 확장 가능한 응용 프로그램에서 CPU, 메모리, 스레드 등의 자원 소비가 최소화되어 있습니다.
